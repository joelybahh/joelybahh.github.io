VR Locomotion Essentials - Unity3D Plugin

An easy to use tool to provide intuitive movement solutions for VR development in Unity3D

New

VR Locomotion Solutions is plugin that handles a variety of locomotion techniques VR! This plugin makes the setup as easy as the click of a button! With only 4 key steps, you can have locomotion in VR working in seconds!

Why would you want this?

well quite simple, this plugin handles logic ranging from headbob movement, arm swinging movement, and pulley movement(used for climbing in VR). Not only this, it also gives the developer alot of options to customize the logic without touching any code! It also handles real-time swapping of locomotion techniques! This means you can go from jogging on the spot in VR to get to a ladder, and then you can simply start climbing the ladder in VR!

Features:

- Real-time locomotion swapping

- Intuitive head-bob locomotion!

- 4 step setup!

- 4 example scenes to get you started!

- Detailed Documentation on everything you need to know about the plugin

- Detailed code comments and function summaries for anyone wanting to dive deap into the plugins logic.
